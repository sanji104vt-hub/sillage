# Sillage サイトポジション移行レポート

## 1. 変更前のサイトポジション

- メンズ香水92本を比較するサイトとして、男性向けの年代・オン／オフを主な入口にしていた。
- トップ、OGP、構造化データ、ブランドページの共通タイトルで「メンズ香水」をサイト全体の対象として表示していた。

## 2. 変更後のサイトポジション

- 性別に限定しない、香水と香りの総合ガイドへ変更した。
- 香調、気分、季節、シーン、価格帯から香水を探せる構成を主軸にした。
- 性別分類は既存フィルターの一つとして維持し、サイト全体の対象を決める条件にはしていない。

## 3. 修正した title と description

### title

- 変更前: `Sillage（シヤージュ）｜メンズ香水比較【2026年版】香調・シーンで選ぶ`
- 変更後: `Sillage｜香調・シーン・季節から選ぶ香水ガイド`

### meta description

- 変更前: メンズ香水92本、20代・30代、オン／オフを中心に説明。
- 変更後: 香調・気分・季節・シーン・価格帯、香水診断、ブランド比較、初心者向けの選び方、香りの時間変化を説明。

## 4. 修正した共通文言

- トップのヒーロー説明。
- トップと商品詳細・記事ページの共通フッター。
- トップのOGP・Twitterカード文言。
- 診断結果シェアカード内のサイト説明。
- フローラル、ウッディ香調の共通説明。
- ItemList名。
- ブランドページの共通title・OGP・Twitter titleから「メンズ人気」を削除。

## 5. 修正しなかった個別メンズ向けコンテンツ

- メンズ商品の固有名称・性別分類。
- メンズ香水を扱う既存記事と比較記事。
- メンズ専門ブランドやメンズ商品のブランド固有説明。
- `men`、`women`、`unisex`の既存フィルターと商品データ。

これらはサイト全体の対象を限定する共通表現ではなく、個別の検索入口・商品事実・記事テーマであるため維持した。

## 6. 診断文言の変更

- 現行の5問と結果文言を確認し、利用者の性別を前提とする質問や結果がないことを確認した。
- 診断ロジックと判定結果は変更していない。

## 7. OGP・構造化データの変更

- トップの `og:title`、`og:description`、`twitter:title`、`twitter:description`を総合香水ガイド向けに変更した。
- `WebSite.description`と`ItemList.name`からサイト全体をメンズへ限定する表現を削除した。
- 商品ページの`Product`、`BreadcrumbList`、価格・評価・在庫には変更していない。
- OGP画像は文字を焼き込んだ専用画像ではないため、画像ファイルを変更していない。

## 8. URL変更の有無

- 商品URL、記事URL、ブランドURL、canonical、sitemap URLの変更なし。
- 商品slug、商品順、性別分類の変更なし。

## 9. SEO上の懸念

- トップの主要検索意図が「メンズ香水比較」から「香水ガイド」へ広がるため、検索結果のtitle・description反映には時間がかかる可能性がある。
- 個別のメンズ記事・商品ページは維持するため、既存のメンズ香水検索意図は個別ページで引き続き受け止める。
- 総合化後も商品構成自体にメンズ分類が多い点は、今後のコンテンツ補完における継続課題となる。

## 10. 公開後に確認すべき検索表示

- Google検索結果のトップtitleとdescription。
- OGP・Xカードのタイトルと説明。
- ブランドページtitleの再クロール状況。
- Search Consoleのクロール、インデックス、検索クエリ変化。
- canonical、構造化データ、sitemapが従来URLを維持していること。
